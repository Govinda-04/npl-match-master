📖 About NPL Match Master

How we built AI-powered predictions for Nepal Premier League cricket

What is NPL Match Master?

NPL Match Master is a machine learning web application that predicts cricket match outcomes for the Nepal Premier League (NPL). It uses real ball-by-ball data from the 2024 and 2025 NPL seasons to train predictive models.

The system provides two core predictions: the likely final score of an ongoing 1st innings, and the winning probability for each team during the 2nd innings chase.

How It Works

1

Data Collection

Ball-by-ball data from 32 NPL 2024 matches covering all deliveries, runs, wickets, and match outcomes.

2

Feature Engineering

Each delivery is transformed into match-state features: current score, run rate, balls remaining, wickets left, and last 5 overs runs.

3

Model Training

Three ML models are trained: XGBoost Regression for score prediction, Logistic Regression and Random Forest for win probability.

4

Live Prediction

Enter the current match state and get instant predictions served through a Flask API with a real-time web interface.

The Models

🎯

XGBoost Regressor

Score Prediction

Gradient boosting model that predicts the final innings total. Trained on 1st innings ball-by-ball data with features like current score, CRR, and balls remaining.

MAE < 5 runs
📈

Logistic Regression

Win Probability

Binary classifier that estimates the probability of the batting team winning the chase. Fast, interpretable, and provides calibrated probabilities.

~80% accuracy
🌲

Random Forest

Win Probability

Ensemble of decision trees that captures complex non-linear relationships between match state and outcome. Robust to overfitting on small datasets.

High precision
